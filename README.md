## math.gsc
###### [IW5](iw5/math.gsc), [T6](t6/math.gsc)
---
#### pow(base, exp)
###### IW5
Returns __base__ to the power of __exp__
```
power = pow(10, 1); // 10
power = pow(4, 5); // 1024
```
---
#### round(num, dec)
###### IW5, T6
Returns __num__ rounded to the __dec__. decimal place
```
rounded = round(4.559, 1); // 4.6
rounded = round(4.559, 2); // 4.56
```
